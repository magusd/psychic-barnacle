## Install
```bash
git clone https://github.com/magusd/psychic-barnacle.git
cd psychic-barnacle
composer install
cp .env.example .env
touch database/database.sqlite 
php artisan migrate --seed
yarn install
```

## TDD

```bash
composer tdd
```

## Run
### Nuxt Server
```bash
yarn start
```
